#  Unity VR Robot Tour Game
Robot Emergy/Andi will give the player a tour on PennState Campus and then an unexpected fire/shooting emergency will happen.

## Prerequisites
* Unity 2021.3.0f1
* VR Hearset (HTC VIVE PRO)
* Controllers (VIVE INDEX)
* Cybershoes
